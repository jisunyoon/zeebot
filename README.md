# zeebot
React, TypeScript, Tailwind CSS를 사용하여 개발된 챗봇 애플리케이션입니다.  <br>
챗봇은 사용자와의 상호작용을 통해 다양한 정보를 제공하거나 간단한 작업을 처리하는 기능을 제공합니다.
